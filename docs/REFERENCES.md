# Ссылки и референсы

Официальная документация и ресурсы для развития движка.

## Рендеринг

- **Ebitengine** — 2D/software рендер: https://github.com/hajimehoshi/ebiten
- **Ebiten Mobile** — gomobile bind для Android/iOS: https://github.com/hajimehoshi/ebiten/wiki/Mobile
- **WebGPU** — GPU рендер: https://github.com/cogentcore/webgpu

## 3D форматы

- **glTF 2.0** — Khronos: https://www.khronos.org/gltf/
- **glTF Sample Models** — тестовые модели: https://github.com/KhronosGroup/glTF-Sample-Models
- **qmuntal/gltf** — Go парсер: https://github.com/qmuntal/gltf

## Физика

- **Bullet** — физический движок: https://github.com/bulletphysics/bullet3
- **Jolt** — Jolt Physics: https://github.com/jrouwe/JoltPhysics

## Инструменты

- **Tauri** — desktop IDE: https://v2.tauri.app/
- **Monaco Editor** — редактор кода: https://microsoft.github.io/monaco-editor/
